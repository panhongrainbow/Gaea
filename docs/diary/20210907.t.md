## 問題和解決方案

### [2021年09月07日 臨時解決方案說明]

在程式碼檔案 Gaea/proxy/router/rule.go，有以下程式碼

```go
func parseRule(cfg *models.Shard) (*BaseRule, error) {
	r := new(BaseRule)
	r.db = cfg.DB // <<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<< 不會被強制改成小寫
	r.table = strings.ToLower(cfg.Table) // <<<<<<<<<<< 都會被強制改成小寫
    // 以下 (略)
```

由以上程式碼，發現數據庫名稱用全小寫

表名可以使用大小寫混合，因為會被程式碼全部強制改成小寫

這樣就很容易觸發解發切片規則
