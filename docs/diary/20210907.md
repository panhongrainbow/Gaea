## 问题和解决方案

### [2021年09月07日 临时解决方案说明]

在程式码档案 Gaea/proxy/router/rule.go，有以下程式码

```go
func parseRule(cfg *models.Shard) (*BaseRule, error) {
	r := new(BaseRule)
	r.db = cfg.DB // <<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<< 不会被强制改成小写
	r.table = strings.ToLower(cfg.Table) // <<<<<<<<<<< 都会被强制改成小写
    // 以下 (略)
```

由以上程式码，发现数据库名称用全小写

表名可以使用大小写混合，因为会被程式码全部强制改成小写

这样就很容易触发解发切片规则
